# The Moribund XML Compendium

A decentralized repository of structural HTML and XML layouts for the **Moribund Architect** ecosystem. 

While our [Theme Preset Compendium](https://github.com/MoribundInstitute/mor-blogger-theme-preset-compendium) handles the CSS "skin", this repository handles the "skeleton" (headers, footers, and layouts).

## 🦴 How It Works

This repository contains raw, unstyled HTML/XML blocks. Because the Moribund Architect desktop app injects dynamic CSS variables (like `var(--theme-bg-panel)`), any structural block you pull from this repo will instantly adapt to your active color theme.
